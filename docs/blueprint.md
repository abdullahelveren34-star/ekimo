# **App Name**: İnsan Kaynakları Merkezi

## Core Features:

- Profile Management: Allow employees to create and manage their profiles, including personal information, skills, and experience.
- Job Postings: Enable HR to post new job openings with detailed descriptions and requirements.
- Application Tracking: System to track job applications and manage the hiring process.
- Document Management: Securely store and manage employee documents, such as contracts, performance reviews, and training records.
- AI-Powered Candidate Screening Tool: AI tool to screen candidate resumes, highlight key qualifications, and match candidates with suitable job postings.

## Style Guidelines:

- Primary color: Deep teal (#008080) to inspire confidence and reliability in the Human Resources function.
- Background color: Light teal (#E0F8F8), which has similar hue as the primary color, but heavily desaturated to 20% to produce a muted backdrop.
- Accent color: Blue-green (#008040), an analogous color with significantly different saturation, which is ideal for interactive elements.
- Font: 'Inter', a grotesque-style sans-serif for a neutral, modern feel suitable for both headlines and body text.
- Use clean and professional icons to represent different HR functions and employee information.
- Design a clear, intuitive layout that facilitates easy navigation and access to key information.
- Incorporate subtle animations to enhance user experience and guide users through different processes.
